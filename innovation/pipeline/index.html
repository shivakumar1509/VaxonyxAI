Pipeline

  1. HLA typing (OptiType / arcasHLA)
  2. Alignment & variant calling (STAR, Mutect2)
  3. Annotation (VEP + key plugins)
  4. Neoantigen prediction (pVACseq; Class I & optional Class II)
  5. Ranking (XGBoost; IEDB/TESLA-informed features)
  6. QC & reporting
